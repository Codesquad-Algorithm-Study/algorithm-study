## 💡 컨벤션

### 파일 이름

`문제이름_문제번호_깃허브아이디` <br>
> ex) 유기농배추_1012_yonghyeonpark

### 커밋 메시지

`문제이름, 시간, 메모리` <br>
> ex) 유기농 배추, 140ms, 16480KB

### 디렉토리 구조

`깃허브 아이디/알고리즘 유형/문제`
> ex) yonghyeonpark/dfs_bfs/Problem.java

### 브랜치 이름

`깃허브 아이디`
